## 구현한 것
- 메인 페이지 디자인  
  - 강의 평가 데이터 DB 조회
- 등록, 신고하기 
  - modal로 구현
- 로그인, 회원가입 페이지 디자인
- 회원가입 및 이메일 인증
- 강의 평가 데이터베이스 구축, 데이터 모델링
- 로그인, 로그아웃 
  - session 구현
- 강의평가 등록, 신고하기 기능
- 미니 검색 엔진
- 추천 및 강의 평가 삭제
- XSS 방어
